Wilderness

About Wilderness

This native app is designed for hiking enthusiasts who are open to discover new hiking routes. The goal is to provide more information about existing trails than most hiking maps do. With more knowledge about the path, users can choose a new route easier.

Time Frame

The app has been created within 5 weeks.

My Role

UX/UI designer, included creating user flow, wireframing, prototyping, user testing, mockups

Tools

  • Adobe XD
  • Photoshop
  • Illustrator
  • Flowmap

Project Overview

The objective was to design a native iOS and Android app for a specific problem. I have chosen to design a native app for those who would like to discover new hiking routes. The problem is that most hikers choose a trail by a hiking map which does not provide all the necessary information for picking an ideal hiking route for their needs. Users might want to choose a route where they can surely find a waterfall or is only few kilometres long and also enjoy-able with kids or is safe for dogs. Wilderness offers a solution for them. Users can create route descriptions themselves and complete them with specific characteristics. Fellow users can comment and rate the descriptions. With the advanced filter options searching for unique trails will be easy. This project has been created as part of my UI programme at CareerFoundry.
